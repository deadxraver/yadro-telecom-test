# Yadro Telecom Test

## Quick Start

To start, simply run `$ make` in terminal, app will be built & tested, 
on success further instructions will appear on the screen.

## Build & Test

To just build app without testing, run `$ make build`,
to run tests, `$ make test`.

## Run

With default configurations you can run example this way:

`$ ./build/task ./test/test.txt`
